Self-Playing Flute

This is my self-playing penny whistle (technically a type of flute). I built this at the end of 2022 and had it play some Christmas songs. Check out some of the videos on the project page!

AI Light-Painter

This is a robot that I built to light paint images for me. As part of this project, I also built a fingerpainting app using the MediaPipe hands model. Users can draw a fingerpainting through hand gestures alone, and then watch as the robot recreates their image perfectly by waving an RGB LED around.
